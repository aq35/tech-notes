(学習)ネットワークアクセスに関する現状

### 把握している事項 ✅

1. **WSL上での通常の通信**
```plaintext
- wsl-vpnkitにより、基本的なネットワークアクセスは可能
- Windows側の証明書とプロキシ設定が自動的に利用される
- apt-get updateや一般的なWebサイトへのcurlが成功
```

2. **Dockerインストールの特殊性**
```plaintext
- Dockerのインストールには公式のGPGキーが必要
- これは通常のWebアクセスとは異なる種類の通信
```
